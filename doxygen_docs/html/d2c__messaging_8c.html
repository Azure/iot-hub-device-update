Submits the message to pending messages store. If the message for specified type already exist, it will be replaced by the new message.

Submits the message to messaging utility queue. If the message for specified type already exist, it will be replaced by the latest message.

Parameters
typeThe message type.
cloudServiceHandleAn opaque pointer to the underlying cloud service handle. By default, this is the handle to an Azure Iot C PnP device client.
messageThe required message content. For example, a JSON string containing 'reported' property of the IoT Hub Device Twin.
responseCallbackA optional callback to be called when the device received a http response.
completedCallbackAn optional callback to be called when the messages processor stopped processing the message.
statusChangedCallbackA optional callback to be called when the messages status has changed.
userDataAn additional user data.
Returns
Returns true if message successfully added to the pending-messages queue.
