bool tagADUC_PnPComponentClient_PropertyUpdate_Context::forceUpdate

Indicates that the property update notification was caused by a client request. For example, when the agent call IoTHub_DeviceClient_LL_GetTwinAsync API. Note: this value should be set to null when calling ClientHandle_SetClientTwinCallback.
