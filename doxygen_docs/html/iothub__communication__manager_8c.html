◆ GetConnTypeFromConnectionString()

ADUC_ConnType GetConnTypeFromConnectionString ( const char *  connectionString)

Scans the connection string and returns the connection type related to the string.

The connection string must use the valid, correct format for the DeviceId and/or the ModuleId e.g. "DeviceId=some-device-id;ModuleId=some-module-id;" If the connection string contains the DeviceId it is an ADUC_ConnType_Device If the connection string contains the DeviceId AND the ModuleId it is an ADUC_ConnType_Module

Parameters
connectionStringthe connection string to scan
Returns
the connection type for connectionString
