Detailed Description

The source update cache for delta download handler update payloads.

Looks up a source update from the source update cache.

Parameters
updateIdProviderThe provider of the UpdateId.
sourceUpdateHashThe hash of the source update that acts as the key for the cache entry.
sourceUpdateAlgorithmThe hash algorithm of the source update that makes up the other half of the composite key.
updateCacheBasePathThe update cache base path. Use NULL for default.
outSourceUpdatePathThe output filepath of the source update in the cache.
Returns
ADUC_Result The result. On success that is not ADUC_Result_Success_Cache_Miss, outSourceUpdatePath will have a path to the source update file.
