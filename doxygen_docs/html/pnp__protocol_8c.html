pnp_protocol.c File Reference

PNP Protocol helper PnP is a convention built on top of existing IoTHub device transport primitives. This header implements functions to aide applications in serializing and deserializing PnP data. Note that actually sending and receiving the data from the underlying IoTHub client is the application's responsibility. More...
