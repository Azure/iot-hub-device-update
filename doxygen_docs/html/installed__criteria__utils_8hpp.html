const bool RemoveInstalledCriteria ( const char *  installedCriteriaFilePath,
const std::string &  installedCriteria 
)

Remove specified installedCriteria from installcriteria data file.

Note: it will remove duplicate installedCriteria entries.
For example, only the baz array element would remain after a call with "bar" installedCriteria:

[
{"installedCriteria": "bar", "state": "installed", "timestamp": "<time 1>"},
{"installedCriteria": "bar", "state": "installed", "timestamp": "<time 2>"},
{"installedCriteria": "baz", "state": "installed", "timestamp": "<time 3>"},
]
Parameters
installedCriteriaFilePathA full path to installed criteria data file.
installedCriteriaAn installed criteria string. Case-sensitive match is used.
Returns
bool 'True' if the specified installedCriteria doesn't exist, file doesn't exist, or removed successfully.
